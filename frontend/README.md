# Retired Tauri frontend

The Tauri settings frontend has been retired. The active settings UI is the WinUI 3 app in `../apps/Azookey.Settings`, which publishes as `settings.exe`.

The candidate window UI is also a WinUI 3 app in `../apps/Azookey.UI` and publishes as `ui.exe`.

This directory is kept only for legacy reference while the migration is cleaned up.
